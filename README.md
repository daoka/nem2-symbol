# nem2-symbol
パッケージインストール
```
npm install
```

ローカル環境での確認
```
ionic serve
```

アカウント作成のブランチ
```
git checkout remotes/origin/create-acount
```

送金のブランチ
```
remotes/origin/send-sym
```
